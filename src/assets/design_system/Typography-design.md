**Typography**

Our design system leverages a purposeful set of typographic styles. We've stress-tested this typographic scale across dozens of projects to make sure it's robust enough to use across (almost) any project, while remaining as accessible as possible for everyone.

## Uso e Diretrizes

- **Display**: para títulos grandes, hero sections e headlines. Usa letter-spacing -0.02em em tamanhos maiores (2xl a md).
- **Text**: para corpo de texto, labels e UI. Letter-spacing 0.
- **Underlined**: variantes para links e elementos interativos com sublinhado.

## Notas e Observações

- **Letter spacing**: no Figma, `-2` corresponde a `-0.02em` em CSS (tracking -2%).
- **Variantes underlined**: `Text xl/Regular underlined` e `Text lg/Medium underlined` — adicionar `text-decoration: underline` ao usar.
